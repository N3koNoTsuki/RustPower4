# RustPower4

Petit projet Rust pour expérimenter les bases (types, constantes, fonctions).

## Structure
- `Power4/src/main.rs` : point d'entrée
- `Power4/src/function.rs` : fonctions métier
- `Power4/src/constants.rs` : constantes
- `Power4/src/types.rs` : types (struct/enum)

## Lancer
```powershell
cargo run
```

## Build
```powershell
cargo build
```
